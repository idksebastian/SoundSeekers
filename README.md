# SoundSeekers

Plataforma web de descubrimiento musical desarrollada con React, Vite, Tailwind CSS y Supabase.

## Tecnologías
- React + Vite
- Tailwind CSS
- Supabase (Auth, Database, Storage)
- React Router DOM

## Ramas
- `develop` — rama de integración principal
- `feature/auth` — autenticación, perfiles y roles
- `feature/home` — homepage, dashboard y reproductor
- `feature/community` — módulo de comunidad
